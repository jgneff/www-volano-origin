You'll need a Java Virtual Machine in order to run the VolanoChat Server, and you'll need a Web Server in order to chat with anyone besides yourself. This page gives you links where you can find both.

Web Servers

If you don't yet have a Web server, you can download and install one of the following to get started:

Unix Apache HTTP Server
Windows O'Reilly WebSite Server
Mac Quarterdeck MacHTTP Server

Or better yet, once you've installed Java you can download Jigsaw regardless of your computer platform. Jigsaw is a full blown HTTP server written entirely in Java by Anselm Baird-Smith at The World Wide Web Consortium.
