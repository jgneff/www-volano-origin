Volano is dedicated to providing unique and entertaining multiuser environments for the Internet community.

6 December 1996 - VolanoChat Version 1.04
VolanoChat Version 1.04 adds host and referrer access control to the Server, along with real-time monitoring capabilities to kick out unwanted guests or permanently ban them from entering. It also includes several performance enhancements to both the Server and Client applet.

1 November 1996 - VolanoChat Version 1.03
VolanoChat Version 1.03 provides standard Java property files for both the Client and the Server, allowing the user to configure the Client user interface dynamically.

4 October 1996 - VolanoChat Version 1.02
VolanoChat Version 1.02 adds standard logging of access, referrer, agent and error information to the Server, allowing Web sites to use the same log analyzer programs for their chat server that they use for their Web server.

6 August 1996 - VolanoChat Version 1.01
VolanoChat Version 1.01 provides advertising capabilities, user profiles, private chat, and many performance enhancements to both the Client and Server.

1 August 1996 - Mark Elderkin joins Volano
Mark Elderkin joins Volano as its Marketing Advisor. Prior to his role at Volano, Mark worked as Director of Marketing and Business Development at RadioMail Corporation, a wireless Internet service provider. He has held strategic marketing and product management positions at CellNet Data Systems, Network Equipment Technologies and National Semiconductor. Mark has a BS in Systems Engineering from Boston University and an MBA from the University of California, Berkeley.

18 June 1996 - VolanoChat Version 1.00
Volano introduces the first release of its Web based chat program, chosen by Motiv Systems as the best among the top 12 Java chat applications for their Java UK Experience site.

14 June 1996 - Volano founded
John Neffenger creates Volano shortly before introducing its first product, VolanoChat Version 1.00. Prior to his role at Volano, John was a software engineer at the IBM Corporation working on Taligent's CommonPoint Application Developer Toolkit for OS/2 Warp. Before his assignment at Taligent, John worked in Palo Alto, California, and Rome, Italy, on IBM's Open Systems Interconnection protocol stack. John has a BA in Mathematics from Northwestern University.
