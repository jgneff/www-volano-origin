Quick tips:

  • Click once on a room name to see who's in the room.
  • Click once on a person's name to see his or her profile.
  • After entering a room, double click on a person's name to pull him or her into a private chat room.
  • If you're in a chat room alone but still want to test out the applet, just double click on your own name to chat privately with yourself.

Adding chat to a Web page

Once you're running the VolanoChat Server at your site, adding chat is as easy as adding a few lines of HTML to a Web page. The HTML applet tag for the VolanoChat Client applet looks like the following, where the parts you modify are shown in italics. 

<applet codebase="http://chat.volano.com/classes"
        archive="COM/volano/VolanoChat.zip"
        code="COM.volano.VolanoChat.class"
        width=500 height=60>
<param name="cabbase" value="COM/volano/VolanoChat.cab">
<param name="color"   value="#ffffff">
<param name="group"   value="Your Room Name">
<param name="text"    value="english.txt">
</applet>

where:

color
is the color you want for VolanoChat's Web page background. You can choose a color to match the background of your Web page. Use the same notation of #RRGGBB as your Web page's bgcolor attribute, where RR is the red hexadecimal value, GG is the green hexadecimal value, and BB is the blue hexadecimal value. The default color is white (#ffffff).
group
is the name of the chat room you want created automatically when someone enters VolanoChat from your Web page. If you omit this parameter, no room for your Web page will be created, but you'll still be able to join other rooms.
text
is the translated text and properties for the applet. The default language is English, specified by english.txt. Instead of English, you can also specify french.txt, german.txt, italian.txt, portuguese.txt, or spanish.txt.

For more information about the HTML applet tag, see the page called Adding an Applet to an HTML Page in the JavaSoft book The Java Tutorial.
