VolanoChat Client and Server $495.00

This software license allows an unlimited number of Client connections from anywhere on the Internet through any Web page on the Internet. You can use it to provide chat at your own Web site or to set up and run your own world wide chat network of Web sites.

VolanoChat is available for immediate electronic delivery through software.net, the Internet software store. Please read the requirements below before purchasing VolanoChat.

Requirements

VolanoChat consists of a Java client applet and a Java server application. The Client applet runs in any Java compatible Web browser, such as Netscape Navigator 3.0 or Microsoft Internet Explorer 3.0. The Server application, though, must run continuously on the computer that hosts your Web site. The Server also requires Java support in the form of a Java Virtual Machine.

If you own and maintain the computer that hosts your Web site, you'll have no trouble setting up and running the VolanoChat Server. If you have only a telnet shell account on the computer that hosts your Web site, you can still run the VolanoChat Server as long as your provider allows you to leave background servers running under your shell account. If you don't run your own Web server, and you don't have a shell account into the computer that hosts your Web site, you probably will not be able to run the VolanoChat Server.

You can find a Java Virtual Machine for your platform from the list of Java implementations recommended by Volano. Also on that page is a list of popular Web servers available over the Internet.
